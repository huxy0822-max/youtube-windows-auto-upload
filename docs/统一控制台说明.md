# 统一控制台说明

以后日常操作，优先使用：

- `dashboard.py`
- 或直接双击 `启动统一控制台.bat`

## 为什么加这个入口

原来有两个 GUI：

- `app.py`：偏单机渲染工作站
- `scheduler_gui.py`：偏批量调度器

现在统一成一个更适合日常跑任务的入口：

- `dashboard.py`

## 新入口能做什么

- 选标签
- 选日期
- 设母带歌曲数
- 一键跑“渲染 + 上传”流水线
- 一键跑“仅渲染”
- 单频道上传
- 查看实时日志
- 统一改工作路径

## 视觉策略

默认开启：

- `全随机视觉`

这会让以下内容按视频随机变化：

- 频谱
- 时间轴
- 黑边
- 缩放
- 配色
- 粒子
- 噪点 / 暗角 / 轻模糊

目的就是减少成片同质化。

## 如果你要精细手调

在 `dashboard.py` 的“高级视觉”页可以手动设参数。

如果你还想用旧界面，也保留了两个入口：

- `app.py`
- `scheduler_gui.py`

## 建议使用方式

日常：

1. 打开统一控制台
2. 选标签、日期
3. 保持“全随机视觉”开启
4. 点“开始流水线”

只有你想细调时，才去高级页或旧版入口。
