# 统一控制台说明

以后日常操作，优先使用：

- `dashboard.py`
- 或直接双击 `启动统一控制台.bat`

## 为什么加这个入口

原来有两个 GUI：

- `app.py`：偏单机渲染工作站
- `scheduler_gui.py`：偏批量调度器

现在统一成一个更适合日常跑任务的入口：

- `dashboard.py`

## 新入口能做什么

- 选标签
- 选日期
- 设母带歌曲数
- 维护多赛道任务清单
- 为多赛道一键建目录
- 预览每个赛道的音乐 / 底图 / 输出路径
- 切换“渲染模块 / 上传模块”
- 一键跑“渲染 + 上传”流水线
- 一键跑“仅渲染”
- 单频道上传
- 按多赛道任务清单批量上传
- 配置文本模型 / 图片模型模板
- 配置网页版那套主提示词、标题库、标题数量、简介长度、缩略图数量
- 直接编辑 `generation_map.json` 的标题、简介、封面、AB 标题
- 保存后自动同步 `upload_manifest.json`
- 查看实时日志
- 统一改工作路径

## 视觉策略

默认开启：

- `全随机视觉`

这会让以下内容按视频随机变化：

- 频谱
- 时间轴
- 黑边
- 缩放
- 配色
- 粒子
- 噪点 / 暗角 / 轻模糊

目的就是减少成片同质化。

## 如果你要精细手调

在 `dashboard.py` 的“高级视觉”页可以手动设参数。

如果你还想用旧界面，也保留了两个入口：

- `app.py`
- `scheduler_gui.py`

## 建议使用方式

日常：

1. 打开统一控制台
2. 在“路径配置”确认音乐目录 / 底图目录 / 输出目录
3. 在“快捷开始”里把今天要跑的 tag 每行填一个
4. 点“批量建目录”，把素材分别放进各赛道目录
5. 在“提示词”页确认 API 模板和内容模板
6. 在“当日内容”页确认标题、简介、封面
7. 回到“快捷开始”选日期
8. 保持“全随机视觉”开启
9. 点“开始当前流程”

如果今天只想上传，不想重渲染：

1. 去“上传”页
2. 保持多赛道任务清单不为空
3. 点“按多赛道任务清单批量上传”

只有你想细调视觉参数时，才去“高级视觉”页或旧版入口。
