# 文件用途说明

## 代码文件

- `app.py`：桌面版渲染工作站主窗口，适合手动点选目录、预览 5 秒效果、启动渲染。
- `daily_scheduler.py`：命令行批量调度器，适合按日期/标签批量跑任务，也能在有外部上传脚本时顺带触发上传。
- `render_engine.py`：最底层的 FFmpeg 调用逻辑，负责检测编码器、拼接音频、渲染视频。
- `scheduler_gui.py`：给 `daily_scheduler.py` 做的一层图形化启动界面。
- `utils.py`：上传侧的公共工具库，负责读配置、解析 `metadata_channels.md`、拿标题/简介/封面、拿浏览器调试端口。
- `browser_api.py`：把浏览器本地接口做成统一层，现在支持 `hubstudio` 和 `bitbrowser` 两种提供方。
- `path_helpers.py`：统一路径解析规则，解决原项目里“当前目录”和“兄弟目录”混用的问题。
- `effects_library.py`：渲染特效参数生成器，负责输出 FFmpeg 的 `filter_complex`。

## 配置文件

- `scheduler_config.json`：调度器的本地运行配置，主要是音乐目录、底图目录、输出目录、上传配置路径、FFmpeg 路径。
- `config/upload_config.json`：上传侧总配置模板，现在也承载浏览器 API 提供方配置。
- `config/channel_mapping.json`：频道映射模板，保存“容器码/浏览器 ID -> 频道信息”的对应关系。

## 文档文件

- `朋友使用教程.md`：原始朋友教程，重点是说明完整上传项目在 macOS + HubStudio 下的使用方式。
- `README.md`：本次整理后的总说明，告诉你当前仓库实际包含什么、不包含什么，以及 Windows/BitBrowser 怎么接。
- `实操配置与提示词说明.md`：Windows + 比特浏览器下的实操配置说明，包含分组策略、提示词修改入口、素材命名、模型切换方式。

## 资源目录

- `fonts/`：渲染时可用的字体资源，给 `drawtext` 和 GUI 预览使用。
- `overlays/`：粒子/天气类覆盖视频，例如雨、雪、萤火虫、尘埃光斑。

## 归档文件

- `归档.zip`：老打包文件，里面主要是 `render_engine.py`、`fonts/`、`overlays/` 的备份，不是完整上传项目。
