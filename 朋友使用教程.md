## 1. 你需要准备什么

### 🖥️ 软件环境

| 软件 | 用途 | 下载地址 |

## 2. 环境安装（一次性）

### Step 1: 安装 Python 依赖

```bash
# 创建虚拟环境（推荐）
cd 上传视频自动化
python3 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install requests playwright

# 安装 Playwright 浏览器引擎
playwright install chromium
```

### Step 2: 验证 HubStudio

1. 打开 HubStudio 客户端
2. 确保 HubStudio API 在本地运行（默认端口 `6873`）
3. 验证连接:

```bash
# 测试 HubStudio API 是否正常
curl http://127.0.0.1:6873/api/v1/env/list -X POST -H "Content-Type: application/json" -d '{"page":1,"size":10}'
```

如果返回 JSON 数据就说明连接成功 ✅

### Step 3: HubStudio 环境设置

在 HubStudio 中，你需要做以下配置：

1. **创建浏览器环境**: 每个 YouTube 频道创建一个独立的浏览器环境
2. **设置标签**: 给同一个项目的频道打上相同标签（如 `大提琴`）
3. **标记 YPP**: 在已加入 YPP 计划的频道备注中加上 `YPP` 字样
4. **登录 YouTube**: 在每个环境中登录对应的 YouTube 账号

> ⚠️ **重要**: 每个环境必须已经登录好 YouTube Studio，脚本不会帮你登录！

## 3. 配置文件修改

### 3.1 修改 `config/upload_config.json`

这是**最重要的配置文件**，你需要改成自己的路径和项目：

```json
{
    "video_folder": "/Users/你的用户名/Downloads/AutoTask",
    "projects_folder": "/Users/你的用户名/你的项目总目录",
    "tag_to_project": {
        "你的标签名": {
            "project_name": "你的项目文件夹名",
            "video_keyword": "你的标签名"
        }
    },
    "ypp_settings": {
        "ab_test_titles": 3,
        "ab_test_thumbnails": 3
    }
}
```

**示例: 假设你做钢琴音乐频道**

```json
{
    "video_folder": "/Users/xiaoming/Downloads/AutoTask",
    "projects_folder": "/Users/xiaoming/youtube-projects/projects",
    "tag_to_project": {
        "钢琴": {
            "project_name": "古典钢琴_治愈系列",
            "video_keyword": "钢琴"
        },
        "吉他": {
            "project_name": "古典吉他_放松精选",
            "video_keyword": "吉他"
        }
    },
    "ypp_settings": {
        "ab_test_titles": 3,
        "ab_test_thumbnails": 3
    }
}
```

### 3.2 修改 `config/channel_mapping.json`

这个文件记录了 **HubStudio 容器码 → 频道** 的映射关系。

第一次使用时，可以先创建一个空的框架，然后运行脚本自动填充：

```json
{
  "version": "1.0",
  "last_updated": "2026-02-15",
  "description": "容器码与YouTube频道名称的映射关系",
  "channels": {
    "你的容器码": {
      "serial_number": 1,
      "tag": "钢琴",
      "channel_name": "你的频道名",
      "channel_id": ""
    }
  }
}
```

> 💡 **容器码在哪里找？** 打开 HubStudio → 环境列表 → 每个环境都有一个唯一的容器码（一串数字）

### 3.3 修改脚本中的硬编码路径

打开 `scripts/batch_upload.py`，找到以下行并改成你自己的路径：

```python
# 第 41 行 - 上传记录保存路径
UPLOAD_RECORDS_DIR = Path("/Users/你的用户名/你的路径/上传记录")
```

### 3.4 播放列表名称映射（可选）

如果你想让视频自动加入播放列表，修改 `batch_upload.py` 中的 `TAG_TO_PLAYLIST`：

```python
TAG_TO_PLAYLIST = {
    "钢琴": "超好聽的鋼琴音樂",
    "吉他": "超好聽的古典吉他音樂",
}
```

## 4. 准备上传材料

### 4.1 视频文件 🎥

视频文件放在 `AutoTask` 文件夹中，**按照以下目录结构**：

```
AutoTask/
├── 0215_钢琴/              ← {日期MMDD}_{标签名}
│   ├── 0215_1.mp4          ← {日期}_{环境序号}.mp4
│   ├── 0215_2.mp4
│   └── 0215_3.mp4
└── 0215_吉他/
    ├── 0215_10.mp4
    └── 0215_11.mp4
```

**命名规则**: `{MMDD}_{HubStudio环境序号}.mp4`

> 💡 每个频道一个视频文件，序号对应 HubStudio 中的环境序号

### 4.2 标题和简介 📝

在项目文件夹中创建 `metadata_channels.md` 文件：

```
你的项目总目录/
└── 古典钢琴_治愈系列/
    ├── metadata_channels.md    ← 标题和简介
    └── images/                 ← 封面图
        ├── 1_01.png
        ├── 2_01.png
        └── 3_01.png
```

**`metadata_channels.md` 格式**：

```markdown
# 频道元数据 - 古典钢琴_治愈系列

---

## 📻 频道 1：你的频道名 (Container: 1)

### 套1
🎹 这里是视频标题

**简介**
```markdown
这里写视频简介内容...
多行简介都可以...
```

### 套2
🎹 第二套备选标题

**简介**
```markdown
第二套简介...
```

---

## 📻 频道 2：第二个频道名 (Container: 2)

### 套1
🎹 第二个频道的标题

**简介**
```markdown
第二个频道的简介...
```
```

**关键说明**：
- `Container: X` 中的数字必须对应 HubStudio 的环境序号
- 每个频道可以准备多套标题/简介（用 `套1`、`套2` 区分）
- YPP 频道需要 **3 套标题**（用于 A/B Testing）
- 非 YPP 频道只需要 **1 套标题**

### 4.3 封面图 🖼️

封面图放在项目文件夹的 `images` 目录下：

```
images/
├── 1_01.png     ← {Container序号}_{套数}.png
├── 1_02.png
├── 1_03.png     ← YPP 频道需要 3 张（A/B Testing）
├── 2_01.png     ← 非 YPP 频道只需要 1 张
└── 3_01.png
```

**封面图要求**:
- 格式: PNG 或 JPG
- 推荐尺寸: 1280 × 720 (16:9)
- 文件大小: 小于 2MB
- 命名格式: `{Container序号}_{套数}.png`

## 5. 运行上传

### 方式一: 自动扫描模式（推荐）

```bash
cd 上传视频自动化/scripts
python3 batch_upload.py
```

脚本会自动扫描 `AutoTask` 文件夹，显示可上传的批次让你选择。

### 方式二: 指定标签和日期

```bash
python3 batch_upload.py --tag 钢琴 --date 2.15
```

### 方式三: 预览模式（不实际上传）

```bash
python3 batch_upload.py --dry-run
```

### 方式四: 只上传某个频道

```bash
python3 batch_upload.py --tag 钢琴 --date 2.15 --channel 1
```

### 运行时会看到什么

```

## 6. 常见问题

### ❓ Q: 脚本运行后显示 "连接 HubStudio 失败"
**A**: 确保 HubStudio 客户端已打开，且 API 端口 `6873` 没有被占用。

### ❓ Q: 上传卡在"选择文件"步骤
**A**: 脚本使用 macOS 的 AppleScript 来操作文件选择对话框，需要给终端 **辅助功能权限**：
- 系统设置 → 隐私与安全 → 辅助功能 → 添加 `终端` 或 `iTerm`

### ❓ Q: 标题/简介没有正确填入
**A**: 检查 `metadata_channels.md` 的格式是否正确，特别是：
- `## 📻 频道 X：频道名 (Container: X)` 这行的格式
- `### 套1` 这行后面直接跟标题
- 简介用 ` ```markdown ` 和 ` ``` ` 包裹

### ❓ Q: 封面图没有上传成功
**A**: 
1. 检查图片命名是否正确: `{Container序号}_{套数}.png`
2. 确保图片在 `images` 目录下
3. 图片不要超过 2MB

### ❓ Q: YPP 频道和普通频道有什么区别？
**A**: 
- **普通频道**: 上传标题 + 简介 + 1张封面 → 直接发布
- **YPP 频道**: 上传 3个标题 + 简介 + 3张封面（A/B Testing）→ 开启收益 → 广告评级 → 发布

### ❓ Q: 脚本只支持 Mac 吗？
**A**: 是的，因为使用了 `osascript`（AppleScript）来处理文件选择对话框。不支持 Windows/Linux。

### ❓ Q: 视频上传失败后可以重传吗？
**A**: 可以。重新运行脚本，已成功的频道会显示 ✅，你可以选择跳过或强制重传。
